Habilidades

HTML (HyperText Markup Language) é uma linguagem de marcação que estrutura o conteúdo das páginas da web, definindo como os elementos (como texto, imagens, links, etc.) são organizados e apresentados. É como a espinha dorsal de um site, fornecendo a base para que outros elementos (como CSS e JavaScript) possam adicionar estilo e interatividade.

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a apresentação visual de documentos HTML, ou seja, como o conteúdo de uma página web será exibido. É responsável por determinar a aparência e o layout de um site, como cores, fontes, espaçamentos e posicionamento dos elementos.

JavaScript é uma linguagem de programação interpretada, de alto nível, amplamente usada para adicionar interatividade e dinamismo a páginas web. É a linguagem padrão para tornar as páginas da web mais do que apenas estáticas, permitindo que elas respondam a ações do usuário, exibam conteúdo dinâmico e interajam com o usuário.

Design Responsivo é uma abordagem de design de sites que permite que o conteúdo se adapte dinamicamente a diferentes tamanhos de tela, proporcionando uma experiência de usuário consistente e otimizada em qualquer dispositivo, seja ele um computador desktop, tablet ou smartphone.
